Calculator Demo

Calculator-js is a standard calculator purely written in Javascript. It is written in a way that can use CSS to change its appearance.

Some of the Calculator's basic functionality has not be implemented yet. This specifically includes:

  • Inverse
  • Negate

These have been disabled on the calculator.

Future Plans

Future development will include the following:

  • A Scientific calculator.
  • The ability to customise the what button are in the calculator.
  • Much more.

Enjoy!
